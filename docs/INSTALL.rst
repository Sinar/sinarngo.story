sinarngo.story Installation
---------------------------

To install sinarngo.story using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``sinarngo.story`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        sinarngo.story

* Re-run buildout, e.g. with:

    $ ./bin/buildout
